# 💰 Precificação Dinâmica de Seguros

> Precificação atuarial dinâmica de seguros com ML. Calcula prêmio justo por perfil de risco usando telemetria, histórico e fatores externos.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/precificacao-seguros/

[![Vertical](https://img.shields.io/badge/Vertical-Finanças-blue)]() [![IA](https://img.shields.io/badge/IA-Machine%20Learning-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**-14%** Loss Ratio · **92%** Precisão · **+22%** Conversão · **R$ 1.240** Prêmio médio

## 🧠 Tecnologias de IA
Machine Learning · GLM · Actuarial

## ✨ O que faz
- Prêmio personalizado por perfil
- Modelo GLM + Gradient Boosting
- Telemetria e fatores externos
- Detecção de fraude em sinistros

## 🗄️ Fonte de dados real
[SUSEP + dados atuariais](https://www.gov.br/susep)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com SUSEP + dados atuariais
2. **Processamento com IA** — Machine Learning · GLM · Actuarial
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
